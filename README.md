
# WoW 3.3.5a Streaming Client Setup

Minimal client that streams MPQs on-demand from a CDN instead of requiring a full local install. The client downloads only what it needs, downloading files in `Data/` as they are accessed ingame, just like in Cataclysm and higher.

---

## How it works

1. Client reads `WoW.mfil` from the game directory on startup
2. MPQs are fetched on-demand via HTTP Range requests from your `source=` URL
3. Downloaded files are cached in `Data/` as `.part` files (resumable across restarts)

---

## Client layout

```
StreamingClient\
│   DivxDecoder.dll
│   Wow.exe
│   WoW.mfil
│
└───WTF\
        Config.wtf
```

**WTF/Config.wtf** minimum:
```
SET realmlist "your.server.address"
SET locale "enUS"
```

Any HTTP server supporting range requests works (nginx, Apache, Caddy, Python `http.server`).
